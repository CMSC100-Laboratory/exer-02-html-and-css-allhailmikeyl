## Steps on how to create GitHub Page
- Starting from the Google Classroom, open the GitHub Classroom provided by the instructor
- Once your repository is initialized, add an "index.html" file
- Once the file is pushed, go to Settings
- Scroll down towards the bottom and set the repository visibility to Public
- Scroll back up, and then, on the left side, find Pages
- Under Build and Deployment, there is a Branch there, select the "main" branch
- Click Save, and wait for a few moments by reloading
- Ta-da! Access the website provided by GitHub ^^
